# Sistema de Controle de Acesso (Portaria)

Sistema desenvolvido em C# para controle de entrada e saída de veículos e visitantes.

## Tecnologias
- C#
- .NET
- Windows Forms
- SQLite
- SQL

## Funcionalidades
- Cadastro de veículos
- Registro de entrada
- Pesquisa por placa ou RG/CPF
- Relatórios em PDF
- Filtro por data
